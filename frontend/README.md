# FinTrack AI — Frontend

This is the React-based frontend for the FinTrack AI project, built with React 19, TypeScript, and Tailwind CSS v4.

## Tech Stack

- **Framework**: React 19 + TypeScript (Vite)
- **Routing**: React Router v7
- **State Management**: 
  - Server state: TanStack Query (v5)
  - Client state: Zustand (for auth and UI preferences)
- **Styling**: Tailwind CSS v4
- **Charts**: Recharts
- **Forms**: React Hook Form + Zod
- **HTTP**: Axios (with interceptors for JWT and refresh)
- **WebSocket**: StompJS + SockJS

## Project Structure

```
frontend/
├── src/
│   ├── api/             # Axios client and endpoint definitions
│   ├── components/      # Shared UI and layout components
│   ├── features/        # Feature-sliced modules (Auth, Dashboard, etc.)
│   ├── hooks/           # Custom hooks (useWebSocket, etc.)
│   ├── lib/             # Utilities and formatting
│   ├── stores/          # Zustand stores
│   ├── App.tsx          # Main routing and app structure
│   └── main.tsx         # Entry point
```

## Setup & Run

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run development server**:
    ```bash
    npm run dev
    ```
    The frontend will be available at `http://localhost:3000`.

3.  **Build for production**:
    ```bash
    npm run build
    ```

## Features

- **Smart Authentication**: JWT-based login, registration, and automatic token refresh.
- **AI Insights**: Personalized spending patterns and recommendations generated by AI.
- **Dynamic Dashboard**: Real-time summary of your finances, recent transactions, and category breakdown.
- **Smart Add (NLP)**: Add transactions using natural language (e.g., "Spent 500 on groceries today").
- **Budget Tracking**: Set limits for categories and receive real-time alerts via WebSockets.
- **Detailed Reports**: Visual charts for monthly trends and spending distribution.
- **Responsive Design**: Modern UI with dark mode support.
