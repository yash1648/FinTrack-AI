# FinTrack AI

## Abstract

The FinTrack AI is an intelligent financial management application designed to help individuals effectively monitor, manage, and analyze their personal financial activities. The system enables users to track income, record expenses, categorize transactions, and visualize financial data through interactive dashboards and reports.

Managing personal finances can be challenging due to a lack of clear insights into spending habits and financial patterns. This project addresses that problem by providing a centralized platform where users can easily record and organize their financial transactions. The application categorizes expenses such as food, transportation, utilities, entertainment, and other daily expenditures, allowing users to better understand where their money is being spent.

A key feature of the system is the integration of artificial intelligence and data analysis techniques to provide intelligent financial insights. The AI component analyzes transaction history to identify spending patterns, detect unusual expenditures, and provide recommendations for better budgeting and savings. The system may also support natural language input, allowing users to add expenses using simple text commands such as "Spent ₹500 on groceries."

The application provides users with visual financial summaries using charts and reports, helping them track their monthly spending, compare expenses across categories, and plan future budgets. It can also generate alerts and recommendations when users exceed their predefined spending limits.

From a technical perspective, the system follows a modern full-stack architecture. The frontend provides a user-friendly interface for interacting with financial data, while the backend manages business logic, data processing, and AI-based analysis. A secure database stores user transactions and financial records.

The FinTrack AI aims to improve financial awareness, encourage responsible spending, and help users make informed financial decisions. By combining financial tracking with intelligent analysis, the system provides a smart and efficient solution for personal financial management.

## Tech Stack

- **Frontend:** React + Vite
- **Desktop:** JavaFX
- **Mobile:** Android
- **Backend:** Spring Boot
- **Database:** PostgreSQL
- **Cache:** Redis
- **AI:** Ollama (Local LLM)
- **Realtime:** WebSocket
